* str[i].compareTo(str[j]) > 0  => i번째 문자가 j번째 문자보다 더 큰수이면, 예를들어 (i = C, j = A 는 성립한다.)

* 문자열의 알파벳 순서를 비교하는 method
* strs[i] 에 저장된 문자열이 strs[j] 에 저장된 문자열보다 알파벳 순서상 뒤에 나오면 결과는 0보다 크다.
